> - 8/23（日）16:00 プラハ着 → プラハ泊
> - 8/24（月） プラハ泊
> - 8/25（火）午前 ウィーンへ移動 → ウィーン泊
> - 8/26（水）夕方 ブダペストへ移動 → ブダペスト泊

## 📍 ① プラハ空港 → 市内（8/23到着時）

### ルートA：エアポートエクスプレス（AEバス）【推奨】

| 項目 | 詳細 |
|------|------|
| 区間 | ヴァーツラフ・ハヴェル空港 → プラハ中央駅（Praha hlavní nádraží） |
| 所要時間 | 約35〜40分 |
| 運行間隔 | 30分に1本（6:30〜22:00） |
| 料金 | 大人 **100 CZK**（約600円）、子供 50 CZK |
| 支払い方法 | バス運転手から現金購入（お釣りが出ない場合あり。小銭持参推奨）または空港チケットカウンター |
| 備考 | 通常の市バス・地下鉄チケットとは別券。16:00着なら乗り継ぎなし直通で非常に便利 |

**注意：** 市内中心部（旧市街・ホテル周辺）への最終アクセスは、プラハ中央駅から地下鉄C線（Hlavní nádraží駅）に乗り換え可能。チケットは共通なので別途購入不要（90分券または24時間券を購入していれば乗り継ぎ可）。

### ルートB：市営バス＋地下鉄【格安】

| 項目 | 詳細 |
|------|------|
| ルート | バス 59番（旧119番） → Nádraží Veleslavín駅 → 地下鉄A線 |
| 所要時間 | バス約20分 + 地下鉄で中心部まで計約45分 |
| 料金 | **90分券 40 CZK**（約250円）/ 24時間券 120 CZK |
| 支払い方法 | 空港到着ホールのPID（公共交通）カウンター・自動券売機（カード可）・バス運転手（現金のみ、やや割高） |
| 備考 | 荷物1個につき20 CZKの荷物料金が別途発生（24時間・72時間券は荷物込み） |

### ルートC：タクシー・Bolt/Uber【快適・割高】

| 項目 | 詳細 |
|------|------|
| 所要時間 | 約30分（渋滞次第） |
| 料金目安 | 約650〜900 CZK（約4,000〜5,500円） |
| 支払い方法 | アプリ経由でカード払い（Bolt・Uberとも利用可） |
| 備考 | 荷物が多い場合・深夜到着に最適。プラハ中央駅のタクシーはぼったくり多発のためアプリ利用を強く推奨 |

### ✅ 8/23（16:00着）の推奨プラン

> **AEバス（100 CZK）** で中央駅へ → 地下鉄で宿泊先最寄駅へ  
> 到着後すぐ動ける22時まで運行のため問題なし。24時間券（120 CZK）を別途買えば翌日の市内移動もそのまま使える。

## 📍 ② プラハ市内 → ウィーン市内（8/25 午前移動）

### 🏆 推奨：Railjet（レイルジェット）/ RegioJet 直通列車

| 項目 | 詳細 |
|------|------|
| 区間 | プラハ中央駅（Praha hlavní nádraží） → ウィーン中央駅（Wien Hauptbahnhof） |
| 運行会社 | **ÖBB Railjet**（オーストリア国鉄）または **RegioJet**（チェコ民間） |
| 所要時間 | 約**4時間**（直通・乗り換え不要） |
| 運行本数 | 1日約10〜14本（直通便）。主要便は2時間ごと |
| 料金 | **€13〜45**（早期予約で€13〜20台も可）。RegioJetは安め |
| 支払い方法 | クレジットカード（オンライン予約推奨） |

#### 予約サイト
- **ÖBB（オーストリア国鉄）公式**：https://www.oebb.at （英語）
- **RegioJet公式**：https://www.regiojet.com （英語）
- **Omio**：https://www.omio.jp （日本語対応・複数社比較可）

#### 午前移動におすすめの便時刻帯

| 出発（プラハ） | 到着（ウィーン） | 備考 |
|----------------|-----------------|------|
| 7:57 / 8:52 | 12:51 / 13:05 | 早めに到着でウィーン観光に余裕あり |
| 9:52 / 10:52 | 13:51 / 14:51 | 午前中の余裕を活かせる |

> ⚠️ **8月はハイシーズン**。人気便は早期完売することも。旅程が確定したら**なるべく早く予約すること**を強く推奨。

#### Railjet vs RegioJet 比較

| | Railjet (ÖBB) | RegioJet |
|---|---|---|
| 車内設備 | ビジネス・1等・2等 | Relax・2等など |
| Wi-Fi | あり | あり |
| 料金 | やや高め | 安め（€13〜） |
| 定時性 | 高い | 概ね良好（遅延有） |
| 推奨度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐☆ |

#### 乗り方メモ
- プラハ中央駅に改札口は**ない**。ホームに直接入れるが、車内検札あり（チケット提示必須）
- 出発直前までホームが決まらないことが多い。電光掲示板を都度確認すること
- 国境越え（チェコ→オーストリア）に特別手続きは不要（シェンゲン協定内）

## 📍 ③ ウィーン市内 → ブダペスト市内（8/26 夕方移動）

### 🏆 推奨：Railjet / EuroCity 直通列車

| 項目 | 詳細 |
|------|------|
| 区間 | ウィーン中央駅（Wien Hauptbahnhof） → ブダペスト東駅（Budapest Keleti） |
| 運行会社 | **ÖBB Railjet** または **EuroCity（EC）** |
| 所要時間 | 約**2時間30分〜40分**（直通） |
| 運行本数 | ほぼ1時間に1本（RailjetとEuroCityが交互） |
| 料金 | **€14〜30**（早期予約で€14〜20台） |
| 支払い方法 | クレジットカード（オンライン予約推奨） |

#### 夕方移動におすすめの便時刻帯

| 出発（ウィーン） | 到着（ブダペスト東） | 備考 |
|-----------------|---------------------|------|
| 15:40 / 16:40 | 18:04 / 19:04 | ブダペスト到着後に夕食・ルインバーへ |
| 17:40 | 20:04 | ウィーン観光を夕方まで楽しめる |

> ⚠️ ブダペスト到着後の夜の行動も考え、遅くとも**18:00発**までを目安にすると余裕あり

#### 乗り方メモ
- ウィーン中央駅は大きく近代的。出発30分前には到着推奨
- ブダペスト東駅（Keleti）到着後、地下鉄M2線・M4線に乗り換え可能
- 1等車利用者は**ÖBB Lounge**（ウィーン中央駅）を無料利用可（6:00〜23:30）

## 📍 ④ ブダペスト市内 → 空港（8/28 20:00出発）

> 20:00発のフライトの場合、**遅くとも18:00には市内を出発**することを推奨（余裕を持って17:30出発が理想）

### ルートA：空港直通バス 100E【推奨】

| 項目 | 詳細 |
|------|------|
| 区間 | Deák Ferenc tér（デアーク広場）→ リスト・フェレンツ国際空港（BUD） |
| 所要時間 | 約30〜35分 |
| 運行間隔 | 日中10分おき、夜間15〜30分おき（24時間運行） |
| 料金 | **2,200 HUF**（約850円）※乗り放題チケット不可・専用料金 |
| 支払い方法 | 運転手から現金（お釣りに注意）または事前アプリ購入 |
| 備考 | 荷物があっても乗りやすい直通便。デアーク広場はブダペスト中心部の交通ハブ |

### ルートB：バス（200E）＋地下鉄M3【格安】

| 項目 | 詳細 |
|------|------|
| ルート | Deák Ferenc tér → M3地下鉄 → Kőbánya-Kispest → バス200E → 空港 |
| 所要時間 | 約50〜60分 |
| 料金 | **24時間券 2,500 HUF** または **72時間券 5,500 HUF**（乗り放題で使える） |
| 備考 | 地下鉄終電に注意。深夜はM3が終電になるためタクシーを推奨 |

### ルートC：タクシー（Bolt/Főtaxi）【快適・確実】

| 項目 | 詳細 |
|------|------|
| 所要時間 | 約20〜30分 |
| 料金目安 | **6,500〜8,000 HUF**（約2,500〜3,100円）、渋滞時はやや上乗せ |
| 支払い方法 | Boltアプリでカード払い（推奨）または現金 |
| 備考 | 荷物が多い場合・最も確実な手段。**Boltアプリ事前DL推奨**（ぼったくり防止） |

### ✅ 8/28（20:00出発）の推奨プラン

> **17:30頃にBolt or 100Eバスで出発** → 18:00〜18:30に空港着 → チェックイン・セキュリティに余裕あり  
> 荷物が多い最終日は**Bolt**が最も楽でトラブルが少ない
